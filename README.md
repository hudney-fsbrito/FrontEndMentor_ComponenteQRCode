### O que aprendi

Consegui assimilar o conceito de posicionamento de imagem e centralização de "div", que era algo que eu vinha com dificuldade.




```css
body {
  height: 100vh;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  text-align: center;
}

.qr-code-image {
  width: 100%;
  box-sizing: border-box;

}
```
